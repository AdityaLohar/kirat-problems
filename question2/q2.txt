Given an array of size N and an integer K, return the count of distinct numbers in all windows of size K. 

Input
The first line of each test case contains a single integer n (size of array) and inetger k (size of window). (1 <= k <= n <= 50)
The second line contains n integers a1, a2, a3 ... an (1 <= ai <= 10000)

Output
Print k integers, distinct numbers in each window of array.

Example
Input:
2
7 4
1 2 1 3 4 2 3

4 2
1 2 4 4 

Output:
3 4 4 3
2 2 1
